# Instructions pour le Carrousel d'Images

## Images Requises

Pour que le carrousel fonctionne correctement, vous devez placer les 3 images suivantes dans ce dossier (`assets/images/`) :

1. **bamako.jpg** - Image de Bamako
2. **senegal.jpg** - Image du Sénégal
3. **plateu.jpg** - Image du Plateau

## Emplacement Actuel des Images

D'après votre capture d'écran, ces images se trouvent actuellement dans un autre dossier.
Veuillez les **copier** dans ce dossier `assets/images/`.

## Vérification

Une fois les images copiées, ce dossier devrait contenir :
- ✅ logo.png (déjà présent)
- ✅ drapeau.png (déjà présent)
- ⚠️ bamako.jpg (à ajouter)
- ⚠️ senegal.jpg (à ajouter)
- ⚠️ plateu.jpg (à ajouter)

## Fonctionnement du Carrousel

Le carrousel est déjà configuré pour :
- Défiler automatiquement toutes les 5 secondes
- Afficher les images en arrière-plan (z-index: -2)
- Utiliser un overlay blanc semi-transparent pour la lisibilité
- Transition douce (fade) entre les images

## Format Recommandé

- **Format** : JPG ou JPEG
- **Résolution** : Minimum 1920x1080 pixels pour une bonne qualité
- **Poids** : Optimisé pour le web (< 500 KB par image si possible)
